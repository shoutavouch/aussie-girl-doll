# Aussie Girl Dolls (Skeleton)

A responsive React + TypeScript storefront skeleton inspired by boutique doll brands. Includes a home page, shop page, product detail view, cart empty state, and about page.

## Features

- Mobileâ€‘responsive layout and styling
- React Router pages: Home, Shop, Product, Cart, About
- Reusable components (header, footer, product card)
- Placeholder product catalog in [src/data/products.ts](src/data/products.ts)

## Development

- Start the dev server: npm run dev
- Build for production: npm run build
- Preview production build: npm run preview

## Where to customize

- Update product data in [src/data/products.ts](src/data/products.ts)
- Edit pages in [src/pages](src/pages)
- Update components in [src/components](src/components)
- Theme and layout styles in [src/index.css](src/index.css) and [src/App.css](src/App.css)
